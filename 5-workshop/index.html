Budget Tracker Workshop — AWS Console Implementation Guide

This workshop provides practical guidance for implementing the Budget Tracker project on AWS, including account setup, IAM, infrastructure, database design, and security configuration.

Module 1: Introduction

Understand the overall architecture of the system, including Route 53, CloudFront, S3, API Gateway, Lambda, DynamoDB, Cognito, SNS/SQS, EventBridge, and CloudWatch.

Module 2: Shared Prerequisites

Set up AWS account and CLI, create a development IAM user, and configure local development tools such as Node.js, .NET 8 SDK, AWS CLI, SAM CLI, and Docker.

Module 3: Foundation Infrastructure

Prepare IAM roles and policies for Lambda, API Gateway, and Cognito, as well as Route 53 DNS and WAF security protections.

Module 4: Database & Data Layer

Create and configure DynamoDB tables, S3 buckets for frontend hosting and receipt storage, and optional lifecycle rules for old receipts.

Module 5: Remaining setup

Continue workshop tasks by deploying required infrastructure, verifying service permissions, and applying security policies.
